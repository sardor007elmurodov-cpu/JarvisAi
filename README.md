## üöÄ Ishga tushirish

JARVISni ishga tushirish uchun quyidagi buyruqni bosing:

1.  **Asosiy Tizim:**
    ```bash
    python start.py
    ```
    Bu JARVISning asosiy interfeysini (HUD), ovozli tizimini va barcha modullarini ishga tushiradi.

2.  **Telegram Bot:**
    Agar siz JARVISni Telegram orqali boshqarmoqchi bo'lsangiz (masofadan):
    ```bash
    python telegram_bot.py
    ```
    Yoki shunchaki `start_telegram_bot.bat` faylini ishga tushiring.

3.  **Veb Boshqaruv (Mobil):**
    Telefondan boshqarish uchun:
    ```bash
    python jarvis_web.py
    ```
    Brauzerda: `http://localhost:5000` manziliga kiring.
